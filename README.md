# autoclass
An automated script capable of running a Google Classroom via Google's APIs

## Quickstart
>[!WARNING]
>Running this script will *not* work if the appropriate credentials are not present.

Quickstart for WSL, Mac, & Linux
```
$ python -m venv .venv
$ source .venv/bin/activate
$ python main.py
```

## Features
- JSON-based description of all items in a course
- Usage of absolute and relative dates to achieve setup flexibility
- Customization for the different kinds of materials present in a given item
- Execution of ideas based on Google's API for Google Classroom and Drive
